# Rotinize

`Análise e Desenvolvimento de Sistemas`

`Projeto: Desenvolvimento de uma Aplicação Móvel em um Ambiente de Negócio`

`2025/2`

O Rotinize é uma aplicação móvel desenvolvida como projeto universitário para facilitar o gerenciamento de tarefas e a consolidação de hábitos diários. Construído com React Native e Expo, o aplicativo oferece uma interface intuitiva para organização pessoal, integrando-se a uma API REST em .NET hospedada no Azure para sincronização de dados.

## Instruções de utilização

Para rodar o projeto no seu dispositivo móvel, siga os passos abaixo:

1. Prepare o ambiente móvel:
- Baixe e instale o aplicativo Expo Go disponível na Google Play Store (Android) ou App Store (iOS).


2. Clone o repositório localmente: Abra seu terminal e execute os comandos:
```bash
git clone https://github.com/ICEI-PUC-Minas-PMV-ADS/pmv-ads-2025-2-e3-proj-mov-t3-rotinize
cd rotinize
cd frontend
```

3. Instale as dependências e inicie:
```bash
npm install
npx expo start
```

4. Execute no celular:
- Abra o app Expo Go no seu celular.
- Utilize a função de câmera do app para escanear o QR Code que aparecerá no seu terminal. O aplicativo será carregado automaticamente.
